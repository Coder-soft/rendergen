RenderGen API

Endpoints Documentation List

API Endpoints

GET /api/animations/{filename}

Retrieve any type of animations from RenderDragon

Example: https://api.ender-dragon.com/dragons
GET /api/fonts/{filename}

Get specific font download from RenderDragon

Example: https://api.ender-dragon.com/dragons/ender
GET /api/music/{filename}

Get any type of music from RenderDragon

Example: https://api.ender-dragon.com/dragons/ender
GET /api/sfx/{filename}

Get any type of sfx from RenderDragon

Example: https://api.render-dragon.com/dragons/ender
